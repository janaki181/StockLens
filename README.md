# 📊 StockLens — Stock Analysis & Comparison System

## 🚀 Overview

**StockLens** is an automated stock analytics system that helps users **analyze and compare companies** using fundamental and technical indicators.

Instead of predicting the market, StockLens focuses on **clarity** — transforming raw stock data into meaningful insights that support better decision-making.

---

## 🎯 Key Features

* **Automated Data Pipeline**

  * Fetches live stock data (NIFTY 50)
  * Processes and stores data in a structured format

* **Data Cleaning & Processing**

  * Handles missing values and inconsistencies
  * Normalizes financial and price data

* **Fundamental Analysis**

  * P/E Ratio
  * ROE
  * Profit Margins
  * Debt-to-Equity

* **Technical Indicators**

  * SMA (20, 50, 200)
  * EMA
  * RSI
  * MACD
  * Bollinger Bands

* **Comparative Analysis**

  * Compare companies across sectors
  * Benchmark against market index (NIFTY)

* **Alerts System**

  * Highlights unusual activity (volume spikes, signals)

* **Interactive Dashboard**

  * Sector-level insights
  * Company deep dives
  * Performance vs NIFTY
  * Visual analytics using Plotly & Dash

---

## 🏗️ System Architecture
